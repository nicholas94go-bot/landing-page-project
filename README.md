# landing-page-project
This project is a foundational course, testing for understanding in introduction to html/css in [theodinproject](https://www.theodinproject.com/paths/foundations/courses/foundations).
The project aim to create a simple webpage _landing page_ in this case and the following files are included:
1. index.html
1. 01.png show final expected outcome of the page.
1. 02.png this image outlines some key details for fonts and colors used.
1. styles.css contains styling applied to the page.

This project is active and is expected to grow in size.
Also feel free to use, modify and contribute
